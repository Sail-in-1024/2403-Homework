术语表 (API)
============

   doc-generatable
      一般的 :term:`可生成文档类型 <doc-generatable>` 须继承自 :class:`py2403homework.utils.Generatable`。
   
   event
      :term:`事件 <event>`，|2403-homework| 发布内容的基本单位，比如作业(:class:`py2403homework.utils.Homework`)。
   
   event type
      :term:`事件类型 <event type>`，一般继承自 :class:`py2403homework.utils.Event`。
