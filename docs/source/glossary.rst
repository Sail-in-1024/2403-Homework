特殊指代
========

:ref:`特殊指代`\ 分为\ :term:`内建特殊指代`\ 和\ :term:`给 伪人 看的`。

以下为\ :term:`内建特殊指代`。

.. glossary::
   
   内建特殊指代
      仅在 :doc:`2403 Homework<index>` 中有特殊含义的指代。
   
   给 *伪人* 看的
      指在 |class-name| 成员中众所周知的指代。
      
      （应该没有人不知道这些吧～）
   
   往期作业记录
      
      - 指在 :doc:`2403 Homework<index>` 项目创建之前布置，但在此后被记录的作业；
      - 指由于 *某些原因* 未能及时发布，在此后补发的作业。
   
   往期事件记录
      
      - 指在 :doc:`2403 Homework<index>` 项目创建之前发生，但在此后被记录的 事件 / 消息；
      - 指由于 *某些原因* 未能及时发布，在此后补发的消息。
      
      类似于 :term:`往期作业记录`\ 。

      这种符号一般有两个意思：
      
         #. 对于部分以数字编号（例如 :title-reference:`《数学作业本》①`）的\ :term:`作业本`，使用此符号用于描述该\ :term:`作业本`\ 的名称；
         #. 对于英语等 **对课时敏感** [#project-sensitive]_ 的作业，使用此符号用于描述课时编号，例如，“英语\ :term:`精彩`\ :term:`③`”表示《精彩三年》中当前单元的第3课时。
   
   ``P``
      “\ **P**\ age”的简称。表示一本练习册中的页码一般用 ``P{数字}`` 或者 ``P{开始}~{结束}``。
   
   803的破电脑
      旧时代的产物。803教室内一台极为老旧的电脑，“803的破电脑”为电脑登录的用户名。

   精彩
      :title-reference:`《精彩三年》` 系列练习本。
   
   自主高效练
      就是语文 :title-reference:`《自主高效练》`。没有别的意思。
   
   知识小测
      科学\ :term:`精彩`\ 的配套练习册书名。
