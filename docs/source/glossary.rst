   ⑩
      这种符号一般有两个意思：
      
         #. 对于部分以数字编号（例如 :title-reference:`《数学作业本》①`）的 :term:`作业本`，使用此符号用于描述该 :term:`作业本` 的名称；
         #. 对于英语等对课时敏感的作业，使用此符号用于描述课时编号。
   
   ``P``
      “Page”的简称。表示一本练习册中的页码一般用 ``P{数字}`` 或者 ``P{开始}~{结束}``。

   精彩
      指 :title-reference:`《精彩三年》` 系列练习本。
   
   自主高效练
